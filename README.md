# FlyerGen

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1QGayP5K_xAi7E1r_8lnfNOUFBM1w1rSy

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app and local API server:
   `npm run server`
   `npm run dev`
3. Open the app settings and save:
   - OpenAI API key for GPT Image 2 image generation and editing
   - Gemini API key for text generation, search, and tags
   - Replicate API key only when using the optional upscaler
